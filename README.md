# Варіант 16 — Сопрограма (Чубко Олег К-26)

## Опис
Репозиторій демонструє реалізацію **варіанта 16**: сопрограма, що читає з стандартного входу дійсні (floating-point) числа і повертає середнє арифметичне трьох останніх значень. Нулі у вхідній послідовності ігноруються, але при надходженні нуля сопрограма **зупиняється** (перехід у паузу) і не враховує цей нуль. Спроба відновити роботу сопрограми раніше ніж за **1 секунду** після отриманого нуля має призвести до відмови (коректної помилки).

> Важливо: суворо не використовуються блокування чи функції `sleep` у самій сопрограмі — перевірка часу здійснюється за допомогою міток часу (non-blocking).

## Файли в репозиторії
- `src/main.py` — реалізація сопрограми та демонстраційна CLI-програма, що читає числа зі стандартного вводу.
- `demo_input.txt` — приклад вхідних даних для демонстрації (можна передавати в програму через перенаправлення).
- `README.md` — цей файл (українською).
- `LICENSE` — ліцензія MIT.
- `.gitignore` — ігнорування тимчасових файлів.
- `screenshots/` — папка з інструкціями, куди додати скріншоти компіляції і роботи (порожні файли-заповнювачі).

## Інструкції — запустити локально
1. Встановіть Python 3.8+ (рекомендовано 3.10+).
2. У терміналі виконайте (в кореневій папці репозиторію):
```bash
python3 -m venv .venv
source .venv/bin/activate   # або .venv\\Scripts\\activate на Windows
python -V
```
3. Запуск програми (читання зі стандартного вводу):
```bash
python src/main.py
```
Після запуску вводьте числа по одному на рядок (натискайте Enter). Для припинення вводу натисніть Ctrl+D (Linux/macOS) або Ctrl+Z та Enter (Windows).

**Альтернативно**, використайте приклад вхідного файлу:
```bash
python src/main.py < demo_input.txt
```

## Пояснення логіки
- Сопрограма реалізована як генератор (PEP 342) — вона приймає значення через `.send()`.
- При отриманні `0.0` вона переходить у стан `paused` та зберігає мітку часу (`time.monotonic()`).
- Якщо клієнт намагається надіслати нове значення раніше ніж за 1 секунду після нуля — сопрограма піднімає `RuntimeError` із комунікативним повідомленням.
- Нулі не враховуються в обчисленні середнього.
- Для кожного надісланого числа (крім нулів), після оновлення буфера останніх трьох значень, сопрограма повертає (yield) поточне середнє трьох останніх (якщо доступні менше трьох — середнє по наявних значеннях).
